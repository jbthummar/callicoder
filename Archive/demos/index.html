Demos of various javascript standalone applications

3D Cone Demo

Demo of 3D Cone.
A 3D cone designed using open source library named 'Cango3D' which provides support for drawing and rotating object in 3D space.
A plane intersecting cone depicts the conic section formed such as "Circle", "Ellipse", "Parabola" and "Hyperbola".
Also user can drag the whole cone using mouse pointers.
Functionality provided for changing cone and plane dimensions.
Plug-ins Used: Cango3D.js

Categorization

Demo of Categorization.
A simple game of categorization is developed in which user has to categorized items in to particular category.
User has to drag the item from the pool and drop it in particular category slot.
A json file is used to store the initial data.
A feedback result based on correct or incorrect attempt is stored below in the table.
Plug-ins Used: jquery.js, jquery-ui.js, Backbone.js
